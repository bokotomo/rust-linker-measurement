# rust-linker-measurement

１行更新のみのビルド時間の測定。  
ライブラリを使うとさらに伸びるっぽい。

## 測定方法

```
cd ./対象のフォルダ

cargo run

ファイル１行変更

cargo run

この時点での数値を測定している。
```
